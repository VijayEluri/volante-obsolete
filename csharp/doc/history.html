Volante : History of Volante

Volante is a continuation of Perst database originally created by Konstantin Knizhnik.

Up to version 2.49 Perst was licensed under BSD. Konstantin (apparently) sold the code to McObject and starting with version 2.50 the license was changed to GPL. McObject sells Perst under a commercial license for people who can't use GPL.

Volante is based on the latest BSD version (2.49) and retains its BSD license which allows free use even in commercial software.

Pers 2.49 was released in 2005 so I used it as an opportunity to make some changes, without worrying about backwards compatibility with Perst.

You can read a detailed changelog but to summarize the changes and why they were made:

  • renamed interfaces to follow C# naming style
  • renamed Storage class to Database
  • excluded XML import/export and replication functionality from official build. My focus is to best support the use case of using Volante as a storage engine for desktop/mobile applications and those
  • excluded old btree code from official build. Perst used to have 2 versions of btree code. I don't know why (possibly for backwards compatibility with old databases) but the new btree code seems to be better (faster) so I removed the old btree. The result is smaller dll and less code to maintain.

My focus with Volante is reliability and documention. This means that most of my work, at least initially, is spent writing documentation, automated test framework, writing more tests and fixing bugs exposed by tests.
