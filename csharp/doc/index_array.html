Volante : Volante Developer's Guide : Persistent array

Persistent array is inteded to be used in classes that use virtual properties to access components of persistent objects. You can't use standard C# array in that case use PArray class instead.

To create persistent array:

  • IPArray<T> CreateArray<T>() where T : class,IPersistent
  • IPArray<T> CreateArray<T>(int initialSize) where T : class,IPersistent

Persistent array implements all the same functionality as link plus:

  • int GetOid(int i)
